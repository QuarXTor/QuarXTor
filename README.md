# QuarXTor

QuarXTor is a hash-graph-based storage stack with RAM-aware deduplication and multi-tier architecture.

This repository is the **meta / top-level** project for the QuarXTor ecosystem:

- Architecture & design docs
- Roadmaps & RFCs
- Specs for QuarXCore / QuarXNet / QuarXDrive / QuarXCloud
- Shared issues & discussions

## Repositories

- [`QuarXTor/QuarXTor`](https://github.com/QuarXTor/QuarXTor) – meta, docs, coordination
- [`QuarXTor/QuarXCore`](https://github.com/QuarXTor/QuarXCore) – core storage engine
- [`QuarXTor/QuarXNet`](https://github.com/QuarXTor/QuarXNet) – networking, RPC, cluster
- [`QuarXTor/QuarXDrive`](https://github.com/QuarXTor/QuarXDrive) – drivers, FUSE, clients
- [`QuarXTor/QuarXCloud`](https://github.com/QuarXTor/QuarXCloud) – cloud / multi-node / orchestration
